Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input LOD into the pop-up and select OK

FNVLODGen

Files to install:

  • Main File - FNVLODGen Resources
  • Optional File - Trees LOD Billboards Vanilla

- Resources required for proper LOD generation

LOD Additions and Improvements

Hide the following:

  • meshes/scol/
    • Hides meshes which are too big for xLODGen to handle
  • textures/clutter/billboard01blod.dds
  • textures/clutter/billboard01blod_n.dds
    • Hides a broken LOD file that would cause some objects to turn green

Running FNVLODGen

  • Click the button on the top bar of MO2
  • Select the FNVLODGen option
  • In the Arguments box, add -o:"FNVLODGen path/Output" and separate from the -fnv argument with a space
    • Make sure to change the path to a file path that actually exists on your computer, for example -o:"C:\Games\Fallout New Vegas Modding\FNVLODGen\Output"
    • Makes sure you keep the -fnv argument as well
  • Select Apply and then exit
  • Run FNVLODGen through MO2
  • In the left pane, right-click and select Select All
  • In the right pane, change both Atlas size options to 8192
  • Check the Terrain LOD option and un-check the Trees LOD option
  • Select Generate
  • Let the generation process run (took around 50 mins on my i5 7500) and close the program once the message LOD Generator finished (you can close this application now) appears
  • Navigate to the file path you put in the Arguments box
  • Add the meshes and textures folder to a single archive and install it through MO2
    • Title the mod FNVLODGen Output
